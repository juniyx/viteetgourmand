Explication des grandes étapes pour faire le deploiement en local de son application web

Dans le cas de l'application Vite&Gourmand le deploiement s'est effectué au moyen du complexe logiciel XAMPP.

Explication des étapes :

1)Installer et lancer XAMPP. Depuis le panneau il faut démarrer les modules : Apache et MySQL

2)Il faut ensuite placer le dossier du code source dans le Dossier "htdocs" du fichier global XAMPP ( le chemin absolu :C:\xampp\htdocs\vite&gourmand\)

3)Puis, en cliquant sur "admin" de MySQL du panneau XAMPP on accède à la plateforme phpmyadmin. c'est l'espace pour la mise en place et la gestion des bases de données et tables.

4)Configurer la connexion à la base de donnée depuis un script php et avec l'utilisation de PDO.

5) Acceder au site en ecrivant dans le navigateur http://localhost/vite&gourmand/ et le script qui est interprété est le "index.php".
